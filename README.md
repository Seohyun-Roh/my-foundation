# My Foundation

### [Demo🐣](https://master--my-foundation-rsh.netlify.app/)

사용해보고 싶었던 기술들을 적용해보고, 만들어보고 싶던 컴포넌트들을 만드는 곳

## 사용 기술

- `React`, `TypeScript`
- `SCSS`, `Styled Components`
- `Recoil`
- `Storybook`
- `i18n`
- `ESLint`, `Prettier`, `Stylelint`

## 프로젝트 설치 & 실행

```bash
git clone https://github.com/Seohyun-Roh/my-foundation.git

yarn

yarn start
```
